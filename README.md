# pingpong
pingpong Game

1. 게임을 하는 사람 이름을 입력하지 않으면 게임을 시작할 수 없음
- P1이름 p2이름 < button >

2. 몇 점 개임인지 설정 가능.
- 이름 입력전에 점수 바꿔야 함.
- 점수를 바꾸면 게임이 0점으로 초기화됨

3. 승리한 사람은 초록색 진사람은 빨간색으로 색 변경 됨
- 듀스 룰 있음

3. 게임이 종료되면 버튼을 누를 수 없지만 승리한 사람의 점수만 -1 할 수 있음(+도 불가 이긴 사람의 -만 가능). 다른 버튼을 누르면 alert

4. reset 버튼 모두 초기화
